**Installation**
-----------------

**To run this app:**
<br/><br/>
Clone down this repository using an IDE such as VS Code.
<br/><br/>
CD into musicSalesApp
<br/><br/>
To run the application type "Yarn run dev", and ctrl + click the link that appears in the terminal.
